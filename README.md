# bialystoker
iOS Tourist app for Bialystok, Poland

This app uses Contentful (https://contentful.com/) as a backend service. All conent stored in Contentful space is based on Wikipedia entries. 

Requirements: 
- Xcode 9.4 
- Carthage https://github.com/Carthage/Carthage  - please use "carthage update --platform iOS --cache-builds" command to build all depencencies
- SwiftLint https://github.com/realm/SwiftLint
